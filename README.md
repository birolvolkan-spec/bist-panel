# Volkan Daily Radar

Telefonla açılabilen, Türkçe ve sade günlük piyasa raporu paneli.

## İçerik

- Kripto: BTC, ETH, SOL, BNB, OP
- Makro: DXY, BTC dominance, USDT dominance, TOTAL/TOTAL2/TOTAL3 mantığı için altyapı
- Emtia: Altın, Brent petrol
- BIST: SASA, ZOREN, BJKAS/BEKTAŞ öncelikli takip
- Günlük fırsat havuzu
- Dün-bugün karşılaştırması
- Pazartesi haftalık özet
- Grafik destekli HTML panel

## Kullanım

1. GitHub Pages açık olmalı.
2. Kaynak olarak `main` branch ve `/root` seçilebilir.
3. `Actions` sekmesinden `Volkan Daily Radar` iş akışı manuel çalıştırılabilir.
4. Panel `index.html` üzerinden açılır.

> Not: Bu sistem yatırım tavsiyesi üretmez. Sadece piyasa takibi ve karar destek panelidir.
